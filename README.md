# 🎬 MoviePilot-Plugins

MoviePilot第三方插件库，提供了一系列实用的插件来增强MoviePilot的功能。

> ⚠️ 注意：本插件库为个人维护，代码结构参考了其他开源项目。推荐优先使用[官方插件库](https://github.com/jxxghp/MoviePilot-Plugins)。
